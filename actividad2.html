Sistema de Detección de Distancia con LED y Alarma

1. Introducción

Este proyecto utiliza un sensor ultrasónico HC-SR04 para medir la distancia a un objeto. La lógica es simple: si la distancia es mayor a 100 cm se enciende un LED; si es menor o igual a 100 cm se activa una alarma (buzzer piezoeléctrico). Se incluye filtrado básico para descartar lecturas inválidas del sensor.

3. Conexiones

  • HC-SR04: VCC → 5V, GND → GND, TRIG → D7, ECHO → D6
  • LED: Ánodo → D2 a través de 220 Ω, cátodo → GND
  • Piezo: (+) → D3, (−) → GND

4. Lógica de funcionamiento

Regla: Si distancia > 100 cm ⇒ LED encendido y alarma apagada. Si distancia ≤ 100 cm ⇒ LED apagado y alarma encendida.

6. Pruebas y resultados

  • LED permaneció encendido con objetos a > 100 cm.
  • Alarma se activó de forma consistente a ≤ 100 cm.
  • El filtrado evitó activaciones falsas por lecturas 0 o > 400 cm.

7. Conclusiones y mejoras

  • Sistema simple y efectivo para alertas de proximidad.
  • Mejoras: promediar N lecturas, tono intermitente o variable, LCD para distancia.
