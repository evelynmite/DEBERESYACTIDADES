Proyecto Arduino: Visualización de Emojis en LCD

Autor: Evelyn Mite

1. Introducción

Este proyecto consistió en la programación de un sistema visual en Arduino que permite mostrar cuatro emojis en forma de carita feliz ubicados en las esquinas de una pantalla LCD de 16x2, generando un efecto de parpadeo sincronizado. La idea principal fue crear una interfaz visual atractiva y dinámica que combine animación con texto fijo en el centro.

El trabajo se desarrolló aplicando conceptos de creación de caracteres personalizados en la librería LiquidCrystal, posicionamiento preciso en coordenadas de pantalla y control de animación mediante alternancia de estados. Todo el proceso permitió poner en práctica conocimientos tanto de electrónica como de programación en C++ para Arduino.

2. Objetivo Específico

El objetivo principal fue diseñar y programar un código que generara un carácter personalizado en forma de emoji, colocándolo en las cuatro esquinas de una pantalla LCD y aplicando un parpadeo simultáneo sin afectar el mensaje central.

Adicionalmente, se buscó reforzar las habilidades en la manipulación de pantallas LCD con Arduino, así como en la creación y almacenamiento de caracteres personalizados, optimizando la lógica para que el efecto visual fuera fluido y constante.

3. Desarrollo

En la primera etapa, se configuró la librería LiquidCrystal para inicializar la pantalla LCD y definir la conexión con los pines del Arduino. Luego, se diseñó la matriz de 5x8 píxeles que dio forma al emoji, la cual fue registrada como carácter personalizado dentro de la memoria del LCD.

Posteriormente, se creó un arreglo con las coordenadas de las esquinas de la pantalla para colocar los emojis de manera precisa. La lógica de parpadeo se implementó mediante una variable booleana que alterna entre mostrar y borrar los caracteres en intervalos de tiempo definidos con la función delay(), logrando un parpadeo uniforme en las cuatro esquinas.

4. Conclusión

El proyecto cumplió con el objetivo de mostrar y animar emojis en una pantalla LCD utilizando Arduino. Se consiguió un efecto visual atractivo manteniendo la estabilidad del texto central, lo que demuestra que es posible combinar información estática con elementos animados en un mismo display.

Además, se comprobó que el uso de caracteres personalizados en LCD es una herramienta versátil para mejorar la estética de proyectos electrónicos, abriendo la posibilidad de implementar íconos, indicadores o elementos decorativos en distintas aplicaciones.

5. Aprendizaje Obtenido

A través de este trabajo se fortalecieron conocimientos sobre el manejo de pantallas LCD con Arduino, especialmente en la creación y control de caracteres personalizados. También se mejoró la comprensión del posicionamiento de elementos gráficos mediante coordenadas.

Se aprendió a optimizar la lógica de parpadeo para que sea fluida y a combinar animaciones simples con texto fijo, logrando un resultado visual más atractivo. Este conocimiento podrá aplicarse en futuros proyectos que requieran interfaces visuales básicas pero funcionales.
