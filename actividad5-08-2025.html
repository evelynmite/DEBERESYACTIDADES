Proyecto: Visualización Dinámica en Pantalla LCD con Arduino

Nombre: Evelyn Marisol Mite Posligua

Fecha: 05 de agosto de 2025

1. Objetivo General

Implementar un sistema con Arduino para mostrar un mensaje en una pantalla LCD de forma dinámica, utilizando funciones de desplazamiento de texto.

2. Materiales Utilizados

  • Arduino UNO
  • Pantalla LCD 16x2
  • Potenciómetro (opcional para contraste)
  • Cables de conexión
  • Protoboard

3. Simulación de actividad


4. Explicación del Funcionamiento

Se utiliza la librería LiquidCrystal para controlar la pantalla LCD. En el setup() se inicializa la pantalla y se imprime un mensaje largo que excede el límite de 16 caracteres por línea. Gracias a la función scrollDisplayLeft(), el mensaje se desplaza dinámicamente hacia la izquierda, generando un efecto de texto en movimiento.

5. Resultados Obtenidos

Al cargar el código en el Arduino, la pantalla muestra el mensaje "hola mi nombre es Evelyn" desplazándose de derecha a izquierda, lo cual permite visualizar mensajes largos de forma clara y atractiva.

6. Conclusiones

  • Se logró mostrar un mensaje dinámico en la pantalla LCD utilizando desplazamiento automático.
  • El efecto visual mejora la experiencia del usuario y la presentación de la información.
  • Este proyecto sirve como base para aplicaciones más complejas con menús, alertas o interacción con el usuario.

7. Recomendaciones

  • Incorporar un potenciómetro para mejorar el contraste visual del LCD.
  • Agregar botones para pausar o reiniciar el desplazamiento del texto.
  • Explorar otras funciones de la librería como scrollDisplayRight() o autoscroll().
